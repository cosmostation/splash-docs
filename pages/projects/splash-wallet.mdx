# Splash Wallet

Designed for the Sui testnet and mainnet! Users can securely manage Sui testnet tokens easily with a seamless and user-friendly experience.

## dApps

dApps are provided through the [dapps.json](https://github.com/cosmostation/splash/blob/main/resources/dapp.json) file.
The file is managed through [Pull Requests](https://github.com/cosmostation/splash/pulls).
